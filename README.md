# NhT BIST 11-Scanner - Android APK

BIST hisse senedi tarama uygulaması. Kivy ile Android'e uyarlanmıştır.

## Scannerlar
- 🚀 Roket Tarama (Vol + CCI + EMA + RSI)
- 🌊 WaveTrend
- 🔷 BOS + CHOCH

## APK İndirme
GitHub Actions sekmesine gidip son başarılı build'den APK'yı indirebilirsiniz.

## Geliştirme
```
pip install kivy pandas numpy yfinance buildozer
```
